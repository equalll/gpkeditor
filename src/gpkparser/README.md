# GPKParser
> markdown parser with peg.js


[Peg.js Source Code](https://github.com/gpkeditor/gpkeditor/blob/master/src/gpkparser/peg-code.peg)

# Install

```bash
yarn add gpkparser
```

# Usage
```javascript

import parser from './parser.js'

console.log(parser.parse('# gpkparser'))

```
